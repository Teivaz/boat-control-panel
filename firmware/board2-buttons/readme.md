# The Board 2. Buttons

This board holds up to 8 buttons with corresponding RGB LEDs for feedback.

The buttons are connected to a port directly and on press shorten the pin to GND.
The RGB LEDs are connected in series from 0 to 7. With 7 not installed.

## Pinout

RA5 Button 0
RA4 Button 1
RA3 Button 2
RA2 Button 3
RA1 Button 4
RA0 Button 5
RA6 Button 6
RA7 Button 7 (not connected)

RC2 RGB LED

RC3 SCL
RC4 SDA

RB0 L/R
RB1 Reserved
